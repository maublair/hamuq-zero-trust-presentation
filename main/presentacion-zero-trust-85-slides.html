Seguridad: 3 Capas de Protección

Capa 1: Autenticación

JWT (JSON Web Tokens) + contraseñas hasheadas con bcrypt. Acceso solo con credenciales válidas.

Capa 2: Autorización

Roles granulares. Cada usuario solo ve/hace lo que le corresponde. Un ejecutor no puede aprobar gastos.

Capa 3: Inmutabilidad (Blockchain)

Cada transacción importante se sella en Polygon. Imposible alterar datos históricos sin dejar evidencia.

Blockchain en Polygon: ¿Qué Significa?

Polygon es una red de blockchain rápida, ecológica y económica.

Rápido: Transacciones en menos de 1 segundo

Ecológico: Usa Proof of Stake (no contamina como Bitcoin)

Económico: Costo por transacción: $0.001 USD aproximadamente

Seguro: Está vinculada a Ethereum, la red más segura del mundo

Cada sello en blockchain = Prueba criptográfica que algo existió en esa fecha/hora y no puede ser modificado.

⚠️ CRÍTICO: Validación MANUAL

El sistema NO valida donaciones automáticamente con el banco.

El encargado de Hamuq debe:

  1. Recibir el comprobante (donante lo sube al sistema)
  2. Verificar en el banco que el dinero llegó realmente
  3. Cargar manualmente en el sistema: fecha, monto, concepto
  4. El sistema cruza datos y detecta discrepancias

Si el encargado no lo hace: no hay datos → no hay alertas → no hay control
