## 📐 REQUISITOS DE DISEÑO ESPECÍFICOS

### CENTRADO PERFECTO:
```
Contenido: Debe estar centrado TANTO horizontal como verticalmente
En el MEDIO del viewport/slider
Sin importar el tamaño del contenido

Fórmula:
- Slideinner: min(1000px, 90vw)
- display: flex
- align-items: center
- justify-content: center
- text-align: center

Si no cabe en un slide:
- Dividir en Slide A + Slide B
- No usar scroll interno
```

### RESPONSIVO:
```
Mobile (< 768px): 
  - Font size: clamp(0.9rem, 3vw, 1.2rem)
  - Cards: 1 columna
  - Espacios reducidos

Tablet (768px - 1024px):
  - Font size: clamp(1rem, 4vw, 1.4rem)
  - Cards: 2 columnas
  - Márgenes medios

Desktop (> 1024px):
  - Font size: 1.1rem base
  - Cards: 3 columnas
  - Márgenes amplios
```

### ANIMACIONES GSAP:
```javascript
// Entrada de slide
gsap.from('.slide-title', {y: -30, opacity: 0, duration: 0.6});
gsap.from('.slide-content', {x: 50, opacity: 0, duration: 0.5, delay: 0.2});

// Hover cards
card.addEventListener('mouseenter', () => {
  gsap.to(card, {scale: 1.05, y: -10, duration: 0.3});
});

// Modal
gsap.fromTo('.modal-box', {scale: 0.8, opacity: 0}, {scale: 1, opacity: 1, duration: 0.4, ease: 'back.out(1.7)'});
```
